# Space Taxi - Roadmap & Ideen

Diese Datei dient als Tracker für die geplanten Verbesserungen und deren Umsetzungsstatus.

## 1. Gameplay & Tiefe
- [ ] **Upgrade-Shop:** Geld ausgeben für Triebwerke, Tank oder Schilde.
- [ ] **Passagier-Varianten:**
    - [ ] VIPs (Trinkgeld abhängig von Flugruhe)
    - [ ] Schwere Aliens (Gewichtsveränderung)
    - [ ] Eilige (Zeitlimit)
- [ ] **Umwelt-Effekte:** Windzonen, Magnetfelder, Asteroidenregen.

## 2. Grafik & Visuals ✅
- [x] **Parallax-Scrolling:** Mehrstufige Hintergründe (Sterne, Nebel, Planeten).
- [x] **Dynamisches Licht:** Triebwerksglühen beleuchtet nahe Oberflächen.
- [x] **Animationen:** Ausfahrende Landebeine kurz vor dem Aufsetzen.
- [x] **Post-Processing:** CRT-Wölbung und Chromatic Aberration.

## 3. Usability & Polish
- [ ] **Interaktives Tutorial:** Einführung der Steuerungsmechanik.
- [ ] **Haptisches Feedback:** Vibrationen bei Mobile-Nutzer.
- [ ] **Highscore-System:** Lokale Bestenliste mit 3-Buchstaben-Signet.

## 4. Immersion & Atmosphäre ✅
- [x] **Weltraum-Funk:** Sektor-Kontrolle Statusmeldungen.
- [x] **Passagier-Avatare:** Pixel-Art Porträts im HUD.
- [x] **Soundscape:** Umgebungs-Ambience je nach Sektor.
